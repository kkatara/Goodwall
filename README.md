# Goodwall

A simple static website prototype for **Goodwall** — a community platform where users can showcase achievements, connect with others, and discover opportunities.

## 📁 Project structure

- `goodwall.html` — main landing page
- `goodwall.css` — stylesheet used by the site
- `Profile/` — profile page(s) for individuals (e.g., `christabel-kisang.html`)

## ▶️ Run locally

1. Open the folder in VS Code.
2. Open `goodwall.html` in your browser.

> Tip: Use the **Live Server** extension in VS Code for auto-refreshing:
> 1. Install **Live Server** (by Ritwick Dey)
> 2. Right-click `goodwall.html` → **Open with Live Server**

## 🛠️ How to contribute

- Edit HTML files in this repo to update content.
- Edit `goodwall.css` to modify styling.
- Add new profiles under `Profile/` and link to them from the main page.

## 📌 Notes

This is a static HTML/CSS project. No build step or server is required.
